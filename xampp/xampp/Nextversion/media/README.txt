# Media Files for Full Page Slider

This directory should contain the following media files for the slider to work properly:

## Video Files
1. video1.mp4 - First video slide
2. video2.mp4 - Third video slide

## Image Files
1. image1.jpg - Second image slide
2. image2.jpg - Fourth image slide

## Requirements
- Videos should be in MP4 format, ideally 1920x1080 resolution or higher
- Images should be in JPG format, ideally 1920x1080 resolution or higher
- All media should have a 16:9 aspect ratio for best results
- Keep file sizes reasonable for web performance (compress if necessary)

## Placeholder Information
Until you add your own media files, the slider will not display properly. Please add your own video and image files with the exact filenames mentioned above.

You can find free stock videos and images at:
- Pexels: https://www.pexels.com/
- Unsplash: https://unsplash.com/
